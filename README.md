## weex-vue-router页面跳转

### 下载

```
git clone https://github.com/heuuLZP/weex-vue-router.git
```

安装依赖

```
npm install
```



### 真机调试

```
weex debug src -e src.index.vue
```



### 注意

index.vue中的url需要修改成你电脑的ip

```
url: 'http://192.168.3.85:8088/weex/new.js',
```





### 文件目录

```
src
├── index.vue    ## 首页
└── new.vue      ## new
```
